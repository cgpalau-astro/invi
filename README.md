# Inverse Time Integration Method (invi)

Functions used in the following papers:

- Constraining the potential of the Milky Way using stellar streams and the Inverse Time Integration method ([`2412.15091`](https://ui.adsabs.harvard.edu/abs/2025MNRAS.539.2718P/abstract))

- Modelling the M68 stellar stream with realistic mass-loss and frequency distributions in angle-action coordinates ([`2508.21408`](https://ui.adsabs.harvard.edu/abs/2026MNRAS.545f1974P/abstract))

Additional installation steps:

Galpy requires GNU Scientific Library (GSL) development package (apt install libgsl-dev)

Install the following python packages: src/agama_v2025 and src/galpy-torusmapper (Within the main folders: python3.11 -m pip install .)

Dustmaps and gaiaunlimited require a manual copy of the files stored in data/
